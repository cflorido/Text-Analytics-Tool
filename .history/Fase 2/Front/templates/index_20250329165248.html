Detectando Noticias Falsas en España

Asegurando información veraz y confiable para todas las personas



Esta aplicación utiliza un algoritmo de clasificación basado en Gradient Boosting para detectar noticias falsas en la política. Los usuarios pueden enviar sus propias noticias en formato de texto o como archivos, y el sistema las analiza utilizando un modelo entrenado con datos verificados. La aplicación también permite reentrenar el modelo directamente, incorporando nuevas noticias de entrenamiento para mejorar su precisión con el tiempo. Para los moderadores de redes sociales, esta herramienta es especialmente útil como un primer filtro, ya que les ayuda a identificar rápidamente contenido potencialmente falso antes de realizar una revisión más detallada, optimizando su trabajo y reduciendo la propagación de desinformación.
